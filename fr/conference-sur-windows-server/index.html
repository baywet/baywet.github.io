Conférence sur Windows Server a SUPINFO Marseille

Dec 15, 2010

Le lundi 17 janvier 2011, le laboratoire Microsoft et les Microsoft Student Partners organisent une conférence sur Windows Server.

Au programme :

·         Windows Server 2008 R2 : Vue d’ensemble

·         Virtualisation avec Hyper-V

·         Nouveautés du Service Pack 1

·         Windows Small Business Server

·         Windows Home Server 2 : Code Name Vail

Speakers:

·         Pierre THURAU

·         Mathieu DUPUY

·         Michael GONZALEZ

Coordinateur: Vincent BIRET

Pour une plus grande facilité dans l’organisation de l’évènement, nous vous invitons à confirmer votre venue ici.
