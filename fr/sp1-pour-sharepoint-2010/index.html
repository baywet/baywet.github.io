SP1 pour SharePoint 2010

Jun 28, 2011

Bonjour,

Ca fait longtemps que je n’ai pas publié d’article (beaucoup de travail en ce moment) alors je poste rapidemment pour vous dire que le SP1 d’office 2010 est sorti (MS voulait le faire coincider avec l’annonce d’Office 365)

Cela signifie donc un SP1 pour SharePoint.

Parmi les nouveautés :

  • Corbeille de collection de sites

  • VSS Writers utilisés pour les déplacements de données lourds

  • Réapparition de storman.aspx qui vous permet de gérer de manière granulaire le l’espace de stockage sur SharePoint

  • Support d’IE9, d’une nouvelle version de chrome et d’autres navigateurs

Plus de détails ici (en anglais)

Pour rappel le processus de mise à jour c’est:

  • SP1 pour foundation

  • SP1 pour les packs de langues installés sur foundation (à passer autant de fois que vous avez de langues installées)

  • SP1 pour SharePoint Server

  • SP1 pour les packs de langues de SharePoint Server installés (à passer autant de fois que vous avez de langues installées)

  • Assistant de configuration des produits sharepoint (afin de mettre à jour les versions déployées)

Tout est décrit plus en détails ici (En Anglais aussi)

Télécharger les WhitePapers (En Anglais)

Voila bonne mise à jour et à bientôt pour de nouveaux articles!
