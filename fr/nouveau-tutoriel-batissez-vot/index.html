Nouveau tutoriel : bâtissez votre solution de provisionnement pour Office 365

Dec 16, 2019

Il y a quelques jours j’ai eu la chance d’animer un atelier avec l’illustre Franck Cornu durant la Modern Workplace Conference Paris.

Nous avons décidé de rentre les labs disponibles via PnP afin que vous puissiez en apprendre plus sur les pratiques et les technologies utilisées.

Si vous suivez les étapes décrites dans les labs, vous en apprendrez sur des technologies telles que Azure Logic Apps, Azure Functions, le Microsoft Graph, Azure Durable Functions, les webhooks SharePoint, PnP Provisioning et plus encore.

N’hésitez pas à ouvrir des issues si quelque chose nécessite une amélioration ou bien même à soumettre des pull requests afin d’améliorer le contenu.

Cliquez ici afin de naviguer vers le repo.

Bons labs !
