    SharePoint 2013 RTM disponible pour les abonnés MSDN!

    Oct 24, 2012 1 min.

    SharePoint 2013 RTM est disponible pour les abonnés MSDN depuis aujourd’hui! C’est un surprise car les communiqués de presse de Microsoft n’annonçaient pas la gamme Office avant mi-Novembre. Le numéro de version officiel est 15.0.4420.1017 Pour l’instant seule la version anglaise est disponible, mais les packs de langues ne devraient pas tarder à arriver.
