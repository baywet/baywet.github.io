    WCF + Entity FrameWork Code First + Azure

    Jul 8, 2012 12 min.

    Introduction Nous allons aujourd’hui nous intéresser à une certaine (ubber) approche de développement qui pourrait convenir à beaucoup. Celle de déployer des WCF auto-générés sur Azure à partir d’un modèle Code First Entity Framework. Mais quelle langue il nous parle là ? Oui en effet quelques rappels s’imposent. Entity Framework : Technologie .NET poussée depuis quelques années pour l’accès et la manipulation des données qui permet de faire abstraction du moteur de bases qui tourne en dessous.
