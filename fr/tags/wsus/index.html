    Mise à jour vers / Update Windows server 2012 et Windows Update Services

    Nov 4, 2012 1 min.

    Petite précision importante tant que je suis dans la mise à jour du parc de serveurs du laboratoire vers Windows Server 2012 : Si vous effectuez une mise à jour (on met le cd et on clique sur « mise à jour ») d’une machine ayant le rôle « Windows Update Services » activée, la mise à jour sera bloquée. En effet la mise à jour de ce rôle n’a pas l’air prise en charge, il vous faudra donc désinstaller WSUS, procéder à la mise à jour et réinstaller/reconfigurer le rôle.
