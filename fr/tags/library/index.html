    Créer des librairies pour les projets SharePoint

    Jun 23, 2012 2 min.

    Récemment au travail on m’a demandé d’extraire du code d’un projet que j’avais réalisé pour gérer la sécurité des formulaires dans SharePoint lors d’un projet client pour qu’il resserve au sein de plusieurs autres projets. Puisqu’il n’y avait « que du code » et pas d’éléments de déploiements propres à SharePoint (ex webparts, contrôles utilisateurs & co) j’ai donc commencé à construire un projet de type librairie de classes afin que les autres développeurs puissent directement inclure cette librairie au sein de leurs projets SharePoint.
