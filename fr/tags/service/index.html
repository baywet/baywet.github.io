    Evolution de la compatibilité système SharePoint 2010 et 2013

    Mar 11, 2014 3 min.

    Le service pack 2 pour SharePoint 2010 et le service pack 1 pour SharePoint 2013 sont récemment sortis. Enfin un support de 2012R2 ! Ils apportent tous les deux le support de windows server 2012R2. En effet cette problématique est récurrente au sein des services informatiques. Comment diminuer le nombre de versions de systèmes d’exploitation en production pour faciliter et industrialiser la maintenance ? Attention cependant la mise à jour « in place » de windows server 2012 vers 2012R2 pour SharePoint 2013 n’est pas supportée par Microsoft.
