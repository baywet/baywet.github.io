    Visual Studio 2013 RC : Comptage des références à votre code

    Oct 5, 2013 1 min.

    J’ai eu l’occasion d’installer Visual Studio 2013 RC récemment. Il apporte plusieurs nouveautés intéressantes. Une de celles qui me plait le plus pour le moment est le comptage de références à votre code. Au-dessus des classes, les propriétés et les méthodes vous avez une indication du nombre de références et en cliquant dessus la liste des références (équivalent à ctrl + shift + f12). Voyez plutôt : Et vous? Avez-vous découvert de nouvelles fonctionnalités intéressantes ?
