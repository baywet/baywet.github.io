Microsoft Build 2020 Microsoft Graph session recordings available

Sep 7, 2020

In case you missed it (and yes I know this post is really late), Microsoft Build 2020 produced hundreds of sessions with great content. The recordings are available on YouTube for free and if you are working with Microsoft Graph, it’s a great way to learn about new features.

I had the opportunity to co-present “What’s new with Microsoft Graph?” with Elise Yang.

Make sure you checkout the Microsoft 365 developer build 2020 playlist.
